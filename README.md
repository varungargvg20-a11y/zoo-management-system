# 🦁 Zoo Management System

A web-based Zoo Management System developed using PHP and MySQL to simplify the management of animal records, zoo operations, and administrative tasks through an easy-to-use interface.

## 📌 Features

- Animal record management
- Add, update, delete, and view records (CRUD)
- Admin dashboard
- Database-driven application
- User-friendly interface
- MySQL database integration

## 🛠 Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## 📂 Project Structure

```
admin/
css/
js/
images/
includes/
index.php
zmsdb.sql
```

## 🚀 Installation

1. Install XAMPP.
2. Copy the project folder into the `htdocs` directory.
3. Import `zmsdb.sql` into phpMyAdmin.
4. Start Apache and MySQL.
5. Open the project in your browser.
